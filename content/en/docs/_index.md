{{% pageinfo %}}
Capa documentation covers the runtime model, Java SDK, integrations, examples, and contribution workflow.
{{% /pageinfo %}}

Start with the [Overview](Overview/) to understand the standard API and pluggable component model. Continue with the getting-started guides and examples for the capability you want to integrate.

The API definitions and implementations evolve independently:

- [cloud-runtimes-jvm](https://github.com/capa-cloud/cloud-runtimes-jvm) defines the JVM runtime APIs.
- [capa-java](https://github.com/capa-cloud/capa-java) implements the Java SDK.
- [capa](https://github.com/capa-cloud/capa) contains the experimental Go sidecar runtime.

Check the status notes in each repository before using a capability in production.
