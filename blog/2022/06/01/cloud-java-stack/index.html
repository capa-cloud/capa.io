Cloud Java Stack

Cloud Java Stack.
Wednesday, June 01, 2022

Capa主要面向Java语言进行设计。

Capa本身不提供功能，而是由底层的各种中间件的Java SDK来提供。

所以了解Java中常用中间件的设计是很有必要的。

本目录主要为对Java生态中中间件的学习和实践。
